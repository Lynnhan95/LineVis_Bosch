How to use?

Data Representation

There are two key KPIs in the assembly line visualization dashboard, Scrap Condition and First Pass Yield, as essential features in measuring the performance of multiple assemly plants and lines.
According to the given dataset, a high scrap rate represents for a bad production performance, while the total production numbers stands for the maximum ability in producing certain kinds of products. Therefore, I choose bar chart and pie chart to visualize each KPI. The same logic could be applied to First Pass Yield.

UI Elements

In order to interact with the dashboard, here are some UI elements to notice:
1.Tags: Use the tags on top of the dashboard to toggle between multiple KPIs.
2.Filter: For example, in Scrap Condition dashboard, there is filter with two options:1(Good Part) with red label and 2(Scrap Part) with blue filter. You may click on each filter to focus on the feature you care about.
3.Shadow: When you hover you mouse over each bar/ pie chart, you will notice a shadow followed, which indicates the area you probably need to focus on
4.Tooltip: Tooltips display the data according to which part you are interacting with, in terms of number, or percentage.
